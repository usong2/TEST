Key Features

Introducing the main features we have prepared for Megazone customers.

  • Social Login

    · User can connect their Google account and use Megazone PoPs with Google login.

  • Single Sign-on

    · Users can sign in to multiple apps with one account.

  • User Management

    · Admin can activate or deactivate organization users

    · Admin can create, edit or delete organization users

    · Admin can grouping organization users

  • Directory Synchronization

    · Admin can synchronize Google Workspace or LDAP Directory users to Megazone PoPs organization

    · Admin can synchronize Google Workspace or LDAP Directory OUs with the Megazone PoPs organization’s groups

  • Security Policy Management

    · Admin can manage 2 step authentication policies

    · Admin can session timeout policies

  • Application Management

    · Admin can assign or unassgin apps to organization users

    · Admin can manage the application’s roles and assign to organization users

  • Event hook

    · Admin can subscribe notifications of events that have occured in Megazone PoPs organization.

    · Admin can subscribe to the events via email, http, or Kakao alimtalk.
